Project Overview

This project combines SQL data cleaning with Power BI to analyze HR metrics such as turnover, hiring rate, and department performance.

The dashboard was designed to simplify the sales monitoring process, allowing sales managers to quickly identify trends, track performance against targets, and spot opportunities for growth.

Key Features

  • Employee turnover trends
  • SQL queries for data wrangling
  • Interactive drill-downs by department
  • Diversity and inclusion tracking

Technologies Used

  • SQL (JOINS,.....)
  • Power Query for data cleansing and transformation
  • Dax
  • Conditional formatting for visual KPIs
  • Data validation for error-proof data entry

Screenshots

Outcome & Impact

This sales dashboard has been implemented by multiple businesses, resulting in an average of 15% reduction in reporting time and improved decision-making capabilities.

Sales managers reported being able to identify underperforming regions and products much faster, allowing for timely interventions that helped improve overall sales performance by up to 12% in some cases.
