Project Overview

This Excel dashboard tracks income, expenses, savings, and debt over time with automated charts and category filters. It includes a dark mode and rolling-year data tracking with dynamic formulas.

Designed for personal finance management, this tracker makes it easy to visualize spending patterns and monitor financial goals with minimal manual input required.

Key Features

  • Dynamic income/expense reports with month-over-month comparison
  • Dark mode toggle for reduced eye strain during night usage
  • Category-level filtering to analyze specific spending patterns
  • Cash flow and savings KPIs with visual goal trackers
  • Automated data validation to prevent input errors
  • Rolling 12-month view with trend analysis

Technologies Used

  • Microsoft Excel (Advanced formulas and functions)
  • Power Query for data transformation
  • Pivot Tables for dynamic reporting
  • Custom VBA macros for automation
  • Conditional formatting for visual indicators
